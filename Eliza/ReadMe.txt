This file consists of code functionality related to two main categories:

1. Pattern Definitions (in 'Patterns' folder): These are the heart of the Eliza chatbot; they consist of the different patterns recognized by the software, the rules for how to process inputted strings from the user, and how to create a response.
2. Analytics: These consist of the following files:                  
                    Eliza: Class consisting of most of the functionality of the program and methods for the functioning of the chatbot
                    ILogger: Interface for a logger for the software
                    IEliza: Interface for the Eliza class (consists of method signatures that are in the primary Eliza class)
                    Memory: Class that implements functionality for conversational memory of the chatbot
